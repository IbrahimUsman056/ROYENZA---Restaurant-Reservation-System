# 🍽️ ROYENZA – Restaurant Reservation System

## Overview

ROYENZA is a full-stack restaurant reservation and table booking platform developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The system enables customers to browse restaurant information, reserve tables online, and submit payment proofs, while administrators can efficiently manage reservations and verify bookings through a dedicated dashboard.

The project aims to streamline the restaurant booking process by providing a modern, user-friendly, and efficient reservation management solution.

---

## Features

### 🍴 Online Table Reservations

* Book tables online in a few simple steps.
* Select reservation details such as date, time, and number of guests.

### 👤 User-Friendly Interface

* Modern and responsive design.
* Seamless experience across desktop and mobile devices.

### 💳 Payment Proof Submission

* Customers can upload payment screenshots after booking.
* Supports reservation confirmation workflow.

### 🛠 Admin Dashboard

* View all reservation requests.
* Review uploaded payment proofs.
* Approve or cancel reservations.
* Manage booking records efficiently.

### 📅 Reservation Management

* Track reservation status.
* Monitor confirmed, pending, and cancelled bookings.

### 🔒 Secure Backend

* RESTful API architecture.
* Secure data handling and database operations.
